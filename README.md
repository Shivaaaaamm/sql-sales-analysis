# 📊 SQL Sales Analysis Project

🚀 SQL-based analysis of retail sales data to uncover key insights related to sales performance, profitability, and customer trends.

---

## 🔍 Business Problem

Businesses often struggle to understand which products and regions drive revenue and profitability, making it difficult to optimize pricing and sales strategies.

---

## 🎯 Objective

To use SQL queries to extract meaningful insights from structured data and support data-driven decision-making.

---

## 📈 Key Insights

* Technology category generates highest sales and profit
* West region performs best across all metrics
* Top customers contribute significantly to total revenue
* Higher discounts negatively impact profitability

---

## 🛠 Tools & Technologies

* SQL (SQLite)
* Data Analysis
* CSV Dataset (Superstore)

---

## 📂 Project Structure

* `analysis.sql` → SQL queries
* `data.csv` → dataset

---

## 💡 Business Impact

This project demonstrates how SQL can be used to analyze business data, identify key trends, and support strategic decision-making.
